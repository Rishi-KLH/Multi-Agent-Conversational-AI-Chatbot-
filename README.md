# 🤖 Multi-Agent Tool-Augmented Chatbot

A multi-agent conversational AI system built using LangGraph, FastAPI, and LLaMA.

## 🚀 Features
- Multi-agent architecture (Planner, Tool, General agents)
- Real-time chat interface
- Wikipedia API for factual queries
- Python REPL for accurate computation
- Dynamic routing and tool invocation

## 🧠 Architecture
- Planner Agent → decides task
- Tool Agent → uses Wikipedia & Python
- General Agent → handles conversation
- LangGraph → manages workflow

## 🛠️ Technologies
Python, FastAPI, LangGraph, LLaMA (Groq), HTML, CSS, JavaScript

## ▶️ How to Run
```bash
pip install -r requirements.txt
python main.py
